About

Hi, I'm Tim! I'm a software-engineer and full-stack web-developer. Before learning to write code and build apps I was a professional touring musician, leading and writing music for a 12-piece band.

I'm interested in both the history and future of music, technology, and especially how they interact with one another.

I've recently graduated from the MakerSquare Advanced Software Engineering Immersive program in Austin, TX

Writing music for an ensemble to perform has a lot of similarites to building an application for the web. I do both! Being part of a team that can make a band succesful, which might include a booking-agent, a manager, and a publicist, is a lot like being a part of a team that can build a successful app, which might include a product-manager, a scrum-master, and software-engineers. I've been a part of both!
